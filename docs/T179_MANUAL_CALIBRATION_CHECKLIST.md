# T179 Manual Calibration Checklist

Use this after adding a fresh OpenRouter key in Praxis Settings.

## One-Time Setup

If the copied journal is inside this repo, run:

```bash
./scripts/use-local-journal.sh
```

This points Praxis at:

`/home/twarga/Documents/praxis/TwargaJournal`

It does not modify your OpenRouter key.

## Re-Analyze Existing English Sessions

Use the app:

1. Open Gallery.
2. Open an English session with a transcript.
3. Click `Re-analyze`.
4. Wait until the session returns to `ready`.
5. Read the Analysis tab.

Start with these copied sessions:

- `2026-05-02_en_take-2` - best long sample.
- `2026-04-26_en_untitled-5` - UI/test sample with enough content.
- `2026-04-27_en_take` - app concept/language-goal sample.

## What To Judge

For each report, check:

- The report is readable, not a tiny bullet summary.
- It does not invent emotional depth for app-test recordings.
- It gives exactly three useful lessons.
- Timestamped moments point to real transcript moments.
- The practice assignment is concrete enough to do before the next recording.
- Language rewrites are natural and useful to practice aloud.
- The scorecard evidence cites actual behavior from the recording.

## Required New Recordings To Complete T179

T179 is not complete until you add fresh v2 analyses across languages:

- 4 English reflective sessions.
- 3 French reflective sessions.
- 3 Spanish reflective sessions.

Each recording should be at least two minutes.

Use this structure:

1. State one topic.
2. Give one concrete example.
3. Explain why it matters.
4. End with one next action.

## Pass Criteria

T179 can be checked only when:

- At least 10 fresh v2 analyses have been reviewed.
- English, French, and Spanish are represented.
- Prose stays in the session language.
- Reports are useful enough that you know what to practice next.
- Any prompt fixes discovered during review have been implemented.
